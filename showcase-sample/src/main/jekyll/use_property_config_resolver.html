Using Properties file configuration resolver

To use properties file Bristleback configuration resolver, you have to define pl.bristleback.server.bristle.conf.resolver.init.PropertiesFileConfigResolver bean and set it as used in Bristleback <bb:servlet> or <bb:standaloneServer> tags.

{% highlight xml %} {% endhighlight %}

You can place your Properties file in project classpath to be recognized by Bristleback. We use Apache Commons Configuration module to resolve Properties file from the classpath.

Details, like the configuration property names, can be found on our wiki page and also in PropertiesFileConfigResolver class documentation.
