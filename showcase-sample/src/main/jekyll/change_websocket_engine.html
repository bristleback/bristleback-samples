Changing Websocket engine

When Websocket engine is not specified, Bristleback uses Jetty Standalone Websocket Engine. This engine is not useful if you want to create web application. To switch to another engine, simply change right property in Bristleback configuration. We will show you how to do it using PojoConfigResolver configuration resolver (if you use PropertiesFileConfigResolver, check Use Property file configuration chapter).

List of the currently supported Websocket engines can be found on Built in Server Engines wiki page. Note that using different engines may require adding new Maven dependencies.
