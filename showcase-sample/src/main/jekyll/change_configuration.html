Changing Bristleback configuration

Bristleback uses initial configuration to resolve all modules used in application. Initial configuration resolver sets various server settings passed by application creator. In this example, we will show how to change configuration using PojoConfigResolver. Just to mention, using good Java IDE, such as Intellij Idea, you can easily list of all available properties.
